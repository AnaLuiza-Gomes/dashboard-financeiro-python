# Dashboard Financeiro em Python

Projeto desenvolvido em Python para análise de dados financeiros e visualização de despesas.

##  Funcionalidades
- Cálculo de receitas, despesas e saldo  
- Análise de dados financeiros  
- Geração de gráfico por categoria  

## Tecnologias utilizadas
- Python  
- Pandas  
- Matplotlib  

## Como executar
1. Faça upload do arquivo financeiro.csv  
2. Execute o script dashboard_financeiro.py  
3. O sistema exibirá o resumo e o gráfico
